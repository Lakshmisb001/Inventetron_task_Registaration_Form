# Registration Form 

## Description
This is a simple Registration Form created using HTML, CSS, and JavaScript.  
It collects user information like First Name, Last Name, Email, Contact, Gender, and Date of Birth.

## Features
- Responsive Registration Form
- Required field validation
- Red star (*) mark for mandatory fields
- JavaScript success alert message
- Clean and simple UI design

## Technologies Used
- HTML5
- CSS3
- JavaScript
